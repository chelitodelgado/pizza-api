# Pizza-API

## Acerca de ...
Pizza-API es un servicio utilizando NodeJS, MongoDB, Express y MongoPaginate. Este es un servicio que 	basico el cuál te permitira realizar ejercicios simple de consultas bajo peticiones en las diferentes tecnologias como JS, Php, Java y framework como Angular, Ionic, React etc.

## Antes de comenzar
Ejecutar el comando "npm install" para instalar las dependencias correspondientes. Una vez instaladas ejecutar "nodemon src/index.js".
